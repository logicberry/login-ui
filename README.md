
# Login Page UI

A responsive Login Ui Page built with flutter
## Tech Stack
Flutter | Dart



## Demo

https://github.com/logicberry/login-ui/blob/main/login_ui.gif

## Run Locally

Clone the project

```
git clone https://github.com/logicberry/login-ui.git
```

Install all dependencies using Flutter:

```
flutter pub get
```

Start the app

```
flutter run
```
